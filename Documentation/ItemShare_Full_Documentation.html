ItemShare for Elder Scrolls Online

ItemShare is an ESO addon that lets players mark tradeable items as shareable in-game, save that data to saved variables, and then import it into a Google Sheet for guild-wide or group-wide request tracking.

ESO Addon Google Sheets Workflow Requires LibCustomMenu Account-wide saved variables

What the Addon Does

ItemShare adds right-click menu options to supported inventory entries so you can quickly add or remove items from your shared list. The addon tracks those items and saves the data for use in an external Google Sheets workflow.

  • Adds right-click options to inventory items allowing you to add or remove them from the shared list
  • Prevents adding items that are bound or temporarily tradeable
  • Displays a visual icon for items that are marked as shared
  • Keeps track of item counts and where items are stored
  • Provides a dedicated window (/itemshare list) showing your shared items
  • Saves all shared data for use in a Google Sheets workflow
  • Allows you to store and send the URL to the shared document to ESO chat
Only items that are actually eligible to be shared can be added. If an item is bound to you, or restricted to temporary group trading, the addon will not present the share option.

Requirements

  • LibCustomMenu is required for the right-click context menu integration.
  • The addon uses the saved variable file ItemShareSavedVars.
  • The companion spreadsheet workflow expects the Google Sheets script files to be included in the GoogleSheets subdirectory of the ZIP package.

Supported Locations

ItemShare can track shared items from multiple storage locations. The location is preserved and exported with the item.

  • Inventory
  • Bank
  • Craft Bag
  • House Storage / Furnishing Vault

Commands

CommandDescription
/itemshareShows command help (debug output).
/itemshare listOpens or closes the Shared Items window.
/itemshare resetClears all saved shared entries.
/itemshare syncReconciles saved entries with items you still own and updates counts and locations.
/itemshare savePrompts to reload the UI so saved variable data is written out.
/itemshare debugToggles debug messages on or off.
/itemshare sheetDisplays the currently configured spreadsheet URL. If a URL is provided, it will be saved.
/itemshare sheet <url>Sets or updates the spreadsheet URL used for sharing.
/itemshare sheet clearClears the stored spreadsheet URL.
/itemshare sharelinkPlaces the configured spreadsheet URL into the chat input for easy sharing.

Alias: /ishare

Spreadsheet Link Integration

ItemShare lets you save your Google Sheets link in-game so you can easily share it with others.

  • You can store your spreadsheet URL using /itemshare sheet <url>
  • The URL will be displayed at the top of the Shared Items window
  • A button in the window allows you to send the URL to chat
  • /itemshare sharelink will prefill the chat box with the URL for easy sharing
The spreadsheet link is provided as plain text. ESO does not support opening external web links directly from the game client.

The Shared Items window (/itemshare list) displays all currently shared items along with their count and location. Right-click a row to toggle sharing on or off. When reopened, only actively shared items are shown.

List Window

The Shared Items window (/itemshare list) shows all items that are currently marked for sharing.

  • Right-click a row to turn sharing off
  • Right-click again to turn sharing back on
  • Left-click the item name to view detailed item information

When the window is closed and reopened, it will only show items that are currently being shared. This is expected behavior.

How Saving Works

ESO only writes saved variables at the appropriate time, so after marking items you should run /ishare save or reload the UI. The addon stores the data in your saved variables file, which can then be processed by the spreadsheet updater.

  1. Mark items in-game using Add to Share.
  2. Run /ishare save or reload the UI.
  3. Locate your saved variables file.
  4. Import that file through the Google Sheets updater.
The saved variables file is typically located in the ESO SavedVariables folder and is named ItemShare.lua or uses the saved variable data for ItemShare. Your spreadsheet updater reads that file and builds the sheets from it.

Spreadsheet Output

The companion Apps Script reads the saved variable data, creates or updates per-account sheets, and rebuilds a master sheet that combines all item sources. The importer keeps the output organized and ready for users to request items.

Typical Workflow

  1. Install the addon and LibCustomMenu.
  2. Mark eligible items in-game as shareable.
  3. Use /ishare save to write the latest data.
  4. Open the Google Sheet that uses the companion Apps Script.
  5. Upload the saved variables file through the spreadsheet updater.
  6. Use the master sheet or account sheet to request items.
  7. Fulfill requests and remove completed items from the sheet as needed.

Notes and Limitations

  • Bound items cannot be shared.
  • BoP group-trade items are intentionally blocked.
  • The spreadsheet side is a separate workflow and must be configured before first use.

Known Issues

  • Marking items in a guild bank as shareable is currently disabled pending further testing
  • If an item is marked as shared while in the Furnishing Vault and is later placed in a house, it will continue to show as shared. This behavior may change in a future update
