ItemShare - ESO Addon

Overview

ItemShare is an Elder Scrolls Online addon that allows players to mark items in-game as shareable and export that information for use in an external Google Sheets tracker.

It is designed for guilds and groups to easily track available items across multiple players and accounts, making item sharing and requesting simple and organized.

What the Addon Does

  • Adds right-click menu options to inventory items
  • Allows marking items as shareable
  • Prevents adding items that are bound or restricted
  • Displays a visual icon for shared items
  • Tracks item count and location
  • Provides a dedicated Shared Items window
  • Saves data for export to Google Sheets
  • Supports sharing a spreadsheet link directly from the addon UI

Requirements

  • LibCustomMenu (Required dependency)
  • Elder Scrolls Online (latest supported API version)

Commands

CommandDescription
/itemshareShows command help (debug output).
/itemshare listOpens or closes the Shared Items window.
/itemshare resetClears all saved shared entries.
/itemshare syncReconciles saved entries with items you still own and updates counts and locations.
/itemshare savePrompts to reload the UI so saved variable data is written out.
/itemshare debugToggles debug messages on or off.
/itemshare guildbankToggles guild bank sharing functionality (currently disabled while under development).
/itemshare sheetDisplays the currently configured spreadsheet URL. If a URL is provided, it will be saved.
/itemshare sheet <url>Sets or updates the spreadsheet URL used for sharing.
/itemshare sheet clearClears the stored spreadsheet URL.
/itemshare sharelinkPlaces the configured spreadsheet URL into the chat input for easy sharing.

Alias: /ishare

Spreadsheet Link Integration

ItemShare allows you to store a reference to your Google Sheets document directly in-game. This makes it easier to share access to your item list with others.

  • You can store your spreadsheet URL using /itemshare sheet <url>
  • The URL will be displayed at the top of the Shared Items window
  • A button in the window allows you to send the URL to chat
  • /itemshare sharelink will prefill the chat box with the URL for easy sharing

Note: The spreadsheet link is provided as plain text. ESO does not support opening external web links directly from the game client.

How It Works

Each item is tracked using a unique key based on:

ItemLink + Location

The addon stores:

  • Item Name and Link
  • Item Type, Quality, and Trait
  • Count
  • Location (Inventory, Bank, etc.)
  • Timestamps

All data is saved using account-wide saved variables and can be exported for external processing.

Typical Workflow

  1. Right-click an item and select Add to Share
  2. Repeat for all items you want to share
  3. Run /ishare save or reload UI
  4. Upload SavedVariables file to Google Sheets
  5. Manage requests and sharing externally

Google Sheets Template & Sample Data

Instead of manually creating and configuring a Google Sheets document, you can use a ready-made template:

Open ItemShare Template Spreadsheet

Click Use Template to create your own copy with the required structure and workflow already configured.

The addon package also includes a SampleData directory containing example SavedVariables files. These can be uploaded to the spreadsheet to see how the system works before using your own data.
