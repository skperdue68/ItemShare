Update ESO Item Sheet

Select your saved variables file and click Update Sheet. The updater will create a worksheet for your ESO @name if it does not already exist, It will list all items shared and copy them to the master sheet as well. RequestedBy values are preserved, and everything is sorted alphabetically.
Suggested Windows location:
Browsers do not allow a web page or Apps Script dialog to auto-select a file from your local Documents folder. You still need to browse to the file yourself.
